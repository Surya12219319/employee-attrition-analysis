📊 Employee Attrition Analysis Dashboard
📌 Project Overview

Employee attrition is a major challenge for organizations because high turnover increases recruitment costs and reduces productivity.
This project analyzes an HR employee dataset to understand attrition patterns and key factors influencing employee turnover.

The goal of this analysis is to help organizations identify trends in employee departures and support data-driven workforce decisions.

This project demonstrates an end-to-end data analytics workflow including:

Data cleaning

Data analysis

KPI calculation

Interactive dashboard visualization

📂 Dataset Information

The dataset contains HR records of employees including demographic details, job information, and attrition status.

🔍 Key Insights

Key observations from the analysis:

Overall employee attrition rate is 16.1%

Research & Development department has the highest number of employees

Attrition patterns vary across age groups and education fields

Attrition is influenced by factors such as job role, experience, and salary

These insights can support organizations in improving employee retention strategies.
